Imprtant FAQ

what is DOM?

The DOM is a programming interface that represents an HTML or XML document as a structured tree, allowing scripts to manipulate its content.

What are DOM events?

DOM events are user or browser actions that trigger code execution, handled using addEventListener().

How to access/modify DOM elements?

Use methods like getElementById() and properties like innerHTML.

How to add/remove DOM elements?

Use createElement(), appendChild(), removeChild().
